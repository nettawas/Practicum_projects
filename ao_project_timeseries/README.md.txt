# Прогнозирование количества заказов такси

Компания «Чётенькое такси» анализирует исторические данные о заказах в аэропортах, чтобы эффективно распределять водителей в часы пиковой нагрузки. Цель проекта — построить модель, предсказывающую количество заказов на следующий час. Целевое качество модели — RMSE не выше 48 на тестовой выборке.

## Описание проекта

Построена регрессионная модель прогнозирования временного ряда с использованием машинного обучения. Применены ресемплирование, генерация признаков и подбор гиперпараметров. Проект завершён успешно: достигнут целевой RMSE менее 48 на тестовых данных.

## Данные

Временной ряд, содержащий количество заказов такси с шагом в 10 минут.  
Данные были ресемплированы по 1 часу и дополнены признаками:

- временные признаки (месяц, день, день недели, час);
- лаговые признаки (значения в предыдущие часы);
- скользящее среднее (короткий тренд).

## Методы и модели

- Предобработка временного ряда: ресемплирование, извлечение признаков;
- Лаги и сглаживание для учёта динамики;
- Обучение моделей: Linear Regression, Decision Tree, Random Forest, CatBoost, LightGBM;
- Подбор гиперпараметров через GridSearchCV;
- Оценка качества по RMSE на обучающей и тестовой выборке.

## Результат

- Лучшая модель: **CatBoost**, RMSE ≈ 44.41
- Альтернатива: **LightGBM**, RMSE ≈ 44.70
- Обе модели соответствуют требуемому уровню качества (RMSE ≤ 48)

CatBoost рекомендован к внедрению: он продемонстрировал наилучший баланс между точностью и обобщающей способностью.  
LightGBM также подходит как надёжное резервное решение.
